CinemaScope

Film ve kısa film platformu olarak geliştirilen bu proje, TMDB API entegrasyonu ile kapsamlı film veritabanı sunmaktadır. Kullanıcıların izleme listesi oluşturmasına ve bağımsız yapımcıların kısa filmlerini paylaşmasına olanak tanıyan web uygulamasıdır.

React TypeScript React Router Axios TMDB API
GitHub

Pre-Pair

Oyun topluluğu için geliştirilmiş eşleştirme platformudur. Kullanıcı kimlik doğrulama sistemi, şifre yönetimi, oyun ekleme ve kullanıcı eşleştirme algoritmaları içeren web tabanlı bir uygulamadır.

Python Flask Firebase HTML CSS
GitHub
